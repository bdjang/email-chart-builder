Instructions

  1. Select a chart type from the drop down menu
  2. Customize the data bar colors, widths, and label text using the customization tools
  3. When you are finished, scroll to the bottom and click on the button to generate your HTML bar chart code
  4. Place into your email template, make adjustments as needed, and test

Tips/Advice

  • For Label Text: to make these bar charts fully mobile responsive, add this snippet of code to your email template:
    @media screen and (max-width: 525px) { .chartMobile td {font-size: 9px !important;} }
    This will reduce the label text when the bar charts are viewed in smaller screen sizes
  • Click on the top üåï emoji to preview dark mode
