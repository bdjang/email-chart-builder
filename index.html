Instructions

  1. Select a chart type from the drop down menu
  2. Customize the bar colors, widths, and label text using the input tools
  3. When you are finished, scroll to the bottom and click on the button to generate your HTML output code
  4. Copy & paste the chart code into your email template, make adjustments as needed, and test

Tips/Advice

  • Use media queries to reduce and adjust the size of the label text for smaller screens. The following snippet of code can be applied to all horizontal and progress bar charts:

    @media screen and (max-width: 525px) { .chartMobile td {font-size: 9px !important;} }

  • The following snippet of code can be applied to all vertical bar charts:

    @media screen and (max-width: 525px) { .chartVertMobile td {font-size: 9px !important; width: 25px !important;} }
